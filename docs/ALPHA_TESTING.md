# Alpha Testing

MOOCs Ultimateは現在、招待した利用者向けのαテスト段階です。東洋大学・INIAD・INIAD MOOCsの公式ツールではありません。

## テスト前の準備

1. 設定ページのバックアップから、必要な設定・メモ・課題状態を書き出す
2. `pnpm install --frozen-lockfile` を実行する
3. `pnpm run release:check` が成功することを確認する
4. `dist/chromium/` をChromeのパッケージ化されていない拡張機能として読み込む

## 重要な注意

- 課題の提出状態と期限表示は補助情報です。最終確認は必ずMOOCs本体で行ってください。
- この拡張機能は課題を代理提出しません。
- 講義資料は授業ルールと著作権を守り、私的利用の範囲で扱ってください。
- APIキーは各テスターが自分の設定ページで入力してください。リポジトリ、Issue、スクリーンショットへ貼らないでください。
- 拡張機能を削除すると`chrome.storage.local`のデータが失われる場合があります。

## 不具合報告

次の情報だけを共有してください。

- 再現手順
- 期待した動作と実際の動作
- ChromeとMOOCs Ultimateのバージョン
- 対象ページの種類（トップ、講義、課題、ACE、Slidesなど）
- APIキー、回答内容、氏名、学籍番号、提出ファイル名を隠したスクリーンショット

診断JSONを添付する場合も、送信前に内容を確認してください。
